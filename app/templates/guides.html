What is PixPlore ?

Pixplore is a web service or application that provides Content-Based Image Retrieval (CBIR) functionality. CBIR is a process used to search for and retrieve images based on their visual content, rather than relying on textual keywords or descriptions.

How to Use PixPlore ?

1. Uploading Your Image Dataset

Users should first upload a dataset of images in the form of a folder containing a collection of images. This image dataset is required before the searching process to have a basis for comparison with the image you want to search for.

2. Inputting the Search Image

Once the dataset is uploaded, users can input an image they want to search for from the dataset.

3. Choosing Search Options

Choose a search option, whether you want to search based on color or texture.

4. Start Searching

Press the search button, and the program will then process, searching for images within the dataset that have similarities to the uploaded image.

5. Viewing Results

The program will display a collection of similar images, sorted from the highest to the lowest similarity. Each image that appears will be accompanied by a percentage indicating its similarity. Information about the number of images displayed and the program's execution time will also be provided.
