What is PixPlore ?

Pixplore is a web service or application that provides Content-Based Image Retrieval (CBIR) functionality. CBIR is a process used to search for and retrieve images based on their visual content, rather than relying on textual keywords or descriptions.

How to Use PixPlore ?

1. Uploading Your Image Dataset

Users should first upload a dataset of images in the form of a folder containing a collection of images. This image dataset is required before the searching process to have a basis for comparison with the image you want to search for.

2. Inputting the Search Image

Once the dataset is uploaded, users can input an image they want to search for from the dataset.

3. Choosing Search Options

Choose a search option, whether you want to search based on color or texture.

4. Start Searching

Press the search button, and the program will then process, searching for images within the dataset that have similarities to the uploaded image.

5. Viewing Results

The program will display a collection of similar images, sorted from the highest to the lowest similarity. Each image that appears will be accompanied by a percentage indicating its similarity. Information about the number of images displayed and the program's execution time will also be provided.

6. Downloading Report

The program generates a report that encompasses the input image, search type, and the visually similar output image. To access the report, simply click the button located at the bottom of the page. If the button is not immediately visible, please be patient for a few moments as the program compiles the report. The time required for this process may vary, especially when dealing with a substantial number of similar images or output.

⚒ Using the Web Scraping Feature

You can use the web scraping feature to add a new dataset. Click on the web scrape button and submit a link to a webpage. Our tool will scan the webpage and extract available images. These images can be added to your dataset, giving you more data to work with for searching and analysis.


Note: Make sure to only scrape websites you have permission to access, and use the images in compliance with their copyright terms.

📽 Using the Camera Feature

You can utilize the camera feature to compare images captured by the camera with those in the dataset. To begin, upload the dataset and then click the "Start" button. Every 10 seconds, Pixplore will capture an image and provide you with visually similar images. Additionally, you have the option to download the report.


Important Note: If you wish to create a dataset by scraping a website, you can start by scraping on the App page. Afterward, proceed to the camera page to utilize the camera feature.
